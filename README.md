# CRM JT Telecom - Frontend

Frontend React/TypeScript do sistema CRM JT Telecom com interface moderna e responsiva.

## 🚀 Tecnologias

- **React 18** - Biblioteca para interfaces de usuário
- **TypeScript** - Superset tipado do JavaScript
- **Vite** - Build tool rápido e moderno
- **Tailwind CSS** - Framework CSS utilitário
- **Shadcn/ui** - Componentes UI modernos
- **React Router** - Roteamento para SPA
- **React Query** - Gerenciamento de estado servidor
- **React Hook Form** - Formulários performáticos
- **Axios** - Cliente HTTP
- **Framer Motion** - Animações fluidas

## 🎨 Design System

- **Cor Primária:** Azul Royal (#4169E1)
- **Cores Secundárias:** Branco (#FFFFFF) e Cinza Claro (#F5F5F5)
- **Tipografia:** Inter/Montserrat/Roboto
- **Framework CSS:** Tailwind CSS

## 📦 Instalação

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm run preview
```

## 🔧 Configuração

1. Copie o arquivo `.env.example` para `.env`:
```bash
cp .env.example .env
```

2. Configure as variáveis de ambiente:
```env
VITE_API_BASE_URL=https://api.app.jttecnologia.com.br
```
